# AFINA MVP - GOBLEXA

AFINA es un MVP desarrollado en Python + Streamlit para análisis financiero inteligente orientado a empresas, contadores y equipos directivos.

El objetivo del proyecto es permitir la carga de archivos financieros, calcular indicadores clave, visualizar KPIs mediante un dashboard, generar diagnósticos asistidos por IA y producir reportes ejecutivos descargables.

## Objetivo del MVP

Este repositorio contiene una versión inicial demostrativa de AFINA, enfocada en validar el flujo principal:

1. Dashboard financiero
2. Carga de archivos Excel / CSV
3. Análisis por industria
4. Proyecciones financieras
5. Chatbot financiero contextualizado
6. Generación de informes PDF / Word
7. Panel administrativo básico

## Tecnologías previstas

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- OpenAI API / LLM
- ReportLab
- python-docx

## Estado del proyecto

Proyecto en etapa inicial de desarrollo MVP.

## Aviso

Este repositorio utiliza datos ficticios o anonimizados para fines demostrativos.  
No se deben subir claves API, archivos financieros reales ni información confidencial.
